# 快速上手

安装完成后，我们就能在 Vue 中引入了

**示例**

```js
import { Button } from "make-wheels";
import Vue from "vue";

new Vue({
  el: "#app",
  components: {
    "j-button": Button,
  },
});
```

**其他组件**

```js
import {
  ButtonGroup,
  Button,
  Icon,
  Input,
  Row,
```
